S-52’s Presentation Library specifies point, line, and area symbols using a compact dialect of HPGL (Hewlett–Packard Graphics Language): pen moves, strokes, arcs, fills, and line styles that ECDIS renderers interpret at runtime.

For this site, the rendered symbols are sourced from the IHO S-101 Portrayal Catalogue rather than executing HPGL instructions. The catalogue provides modern assets (including day/dusk/night variants) and metadata that we map to the expected symbol identifiers. See the [S-101 Portrayal Catalogue](https://github.com/iho-ohi/S-101_Portrayal-Catalogue.git).
